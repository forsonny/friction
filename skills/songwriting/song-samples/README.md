# Song Samples Library

Real-world examples of songs that exemplify the Friction methodology. Use these as reference when writing.

## Purpose

These analyses show how successful songs:
- Use the 7 structure types effectively
- Open without time-of-day cliches
- Create hooks that sound said, not written
- Avoid genre-specific cliches
- Build conversational texture

## How to Use

1. **Before writing**: Read 2-3 analyses in your target genre
2. **When stuck on structure**: Find a song with similar emotional arc
3. **When stuck on opening**: See how different songs enter the story
4. **When stuck on hook**: Study what makes successful hooks conversational
5. **When reviewing**: Compare your draft against these examples

## Analysis Format

Each song analysis includes:
- **Structure Analysis**: Which of the 7 types, section breakdown
- **Opening Technique**: Which of the 8 techniques
- **Hook Analysis**: What makes it work, syllable count
- **Notable Techniques**: Specific craft choices
- **"Said Not Written" Lines**: Examples of conversational quality
- **Cliche Avoidance**: What they didn't do

## Selection Criteria

Songs were selected for:
1. Commercial or critical success
2. Strong craft (not just popularity)
3. Avoidance of genre cliches
4. Memorable, quotable lines
5. Interesting structural choices
